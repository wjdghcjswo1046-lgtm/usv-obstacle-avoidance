
# 자율운항 보트 장애물 회피 알고리즘 (ROS2)

교내 자율운항선박 경진대회에서 사용한 장애물 회피 알고리즘 코드입니다.
센서 데이터를 기반으로 장애물을 탐지하고, 회피 경로를 선택한 뒤 조향 및 추진기 제어까지 연결하도록 구성했습니다.

## 1. 프로젝트 개요

* 대회: 교내 자율운항선박 경진대회 (장려상)
* 역할: 팀장 및 전체 총괄, 장애물 회피 알고리즘 구현
* 환경: ROS2 기반 자율운항 보트

## 2. 사용 센서 및 제어 대상

* 입력 센서

  * LiDAR (`/scan`)
  * IMU (`/imu/data`)
  * GPS (`/gps/fix`)
* 출력 제어

  * 추진기 출력 (`/actuator/thruster/percentage`)
  * 조향각(Key degree) (`/actuator/key/degree`)

## 3. 코드 구성 (핵심 함수)

* `imu_callback()`

  * IMU 쿼터니언을 헤딩(Yaw)으로 변환하고 보정 헤딩을 계산
* `gps_callback()` / `check_waypoint_reached()`

  * 현재 위치 업데이트 및 웨이포인트 도달 여부 확인
* `lidar_callback()`

  * LiDAR 데이터 전처리 → 장애물 탐지 → 회피 경로 계산 호출
* `calculate_path()`

  * 장애물 개수에 따라 단일/복수 장애물 회피 전략 분기
* `avoid_single_obstacle()`

  * 단일 장애물 기준 좌/우 회피 경로 비교 후 선택
* `avoid_obstacles()`

  * 복수 장애물 사이 경로 후보를 비교해 최적 회피 경로 선택
* `navigate_to_angle()`

  * 선택된 목표 각도에 따라 조향/추진 제어

## 4. 장애물 회피 알고리즘 흐름

1. **LiDAR 데이터 수신**

   * `/scan` 토픽으로부터 거리 데이터를 수신합니다.

2. **거리값 전처리**

   * 거리값 범위를 제한(clipping)하여 비정상값 영향을 줄입니다.
   * 중앙값 필터(median filter)를 적용해 스파이크 노이즈를 완화합니다.

3. **장애물 후보 추출**

   * 유효 거리 조건을 만족하는 포인트만 사용합니다.
   * LiDAR 극좌표 데이터를 Cartesian 좌표로 변환한 뒤, DBSCAN으로 장애물 군집을 분리합니다.
   * 각 군집의 평균 좌표를 대표 장애물 위치로 사용합니다.

4. **장애물 개수에 따른 회피 전략 분기**

   * 장애물이 1개인 경우: 좌/우 회피 경로 후보를 비교합니다.
   * 장애물이 2개 이상인 경우: 장애물 사이 통과 가능 구간(gap) 후보를 평가합니다.

5. **목표 회피 각도 결정**

   * 단일 장애물: 웨이포인트 방향과의 각도 차이가 더 작은 경로를 선택합니다.
   * 복수 장애물: 안전 여유와 변침 부담을 함께 고려한 가중치 기반으로 최적 gap의 중심 각도를 선택합니다.

6. **조향각 및 추진기 출력 제어**

   * 선택된 목표 각도를 기반으로 조향각(Key degree)을 단계적으로 설정합니다.
   * 단일/복수 장애물 상황에 따라 추진기 목표 속도를 조정하여 안정적으로 회피 기동을 수행합니다.

## 4-1. 장애물 회피 계산 방식 (간략)

본 코드에서는 장애물 수에 따라 회피 로직을 다르게 적용하며, 특히 **복수 장애물 상황에서 경로 선택 기준**을 명시적으로 두었습니다.

### 1) 단일 장애물 회피

* 장애물 중심 각도를 기준으로 **좌측 회피 경로**와 **우측 회피 경로**를 생성합니다.
* 각 경로 중심각과 **웨이포인트 방향(angle to waypoint)** 의 차이를 비교하여, 목표 진행 방향과 더 잘 맞는 경로를 선택합니다.
* 각도 차이가 동일한 경우에는 웨이포인트가 위치한 방향(좌/우)과 같은 쪽 경로를 우선 선택합니다.

### 2) 복수 장애물 회피

* 장애물 군집들의 중심 각도를 정렬한 뒤, 인접 장애물 사이의 **각도 간격(gap)** 을 계산합니다.

* 각도 간격이 안전 기준(safety angle)보다 큰 경우에만 통과 가능 경로 후보로 사용합니다.

* 각 후보 gap에 대해 다음 요소를 반영하여 가중치를 계산합니다.

* **가중치 A (안전 여유 반영)**

  * gap이 넓을수록 높은 점수를 부여합니다.
  * 즉, 장애물 사이 여유가 큰 경로를 우선 고려합니다.

* **가중치 B (기동 부담 반영)**

  * 현재 헤딩과 gap 중심각의 차이가 작을수록 높은 점수를 부여합니다.
  * 즉, 급격한 선회보다 현재 진행 방향에서 무리 없이 전환 가능한 경로를 우선 고려합니다.

* **최종 경로 선택**

  * 결합 가중치(가중치 A + 가중치 B)가 가장 큰 gap의 중심 각도를 최종 회피 목표 각도로 선택합니다.

### 3) 회피 제어 출력

* 최종 선택된 목표 각도에 따라 조향각(Key degree)을 단계적으로 설정합니다.
* 장애물 상황(단일/복수)에 따라 추진기 목표 속도를 낮춰 회피 안정성을 확보하도록 구성했습니다.
